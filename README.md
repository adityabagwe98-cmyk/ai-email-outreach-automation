# AI-Powered Email Outreach Automation

## 📌 Overview

This project is an AI-powered email outreach automation built with **n8n**, **Groq LLM**, **Google Sheets**, and **Gmail**.

The workflow automatically reads customer data from Google Sheets, generates personalized outreach emails using AI, sends the emails through Gmail, and updates the lead status after successful delivery.

---

## 🚀 Workflow

Google Sheets (Read Leads)
↓
AI Agent
↓
Groq LLM
↓
Gmail
↓
Google Sheets (Update Status)

---

## ✨ Features

* Reads lead data from Google Sheets
* Generates personalized outreach emails using AI
* Sends emails automatically via Gmail
* Updates lead status from **Pending** to **Done**
* Eliminates repetitive manual work
